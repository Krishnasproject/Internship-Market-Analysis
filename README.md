
📊 Internship Market Analysis Dashboard
An end-to-end data analysis project that scrapes internship data from Internshala, cleans and processes it using Python, and visualizes insights through an interactive Tableau dashboard.

🚀 Project Workflow
Web Scraping using requests & BeautifulSoup
Data Cleaning & Transformation using pandas
Exploratory Data Analysis (EDA)
Interactive Dashboard Creation in Tableau
🛠️ Tech Stack
Python (requests, BeautifulSoup, pandas)
Jupyter Notebook
Tableau
📂 Project Files
app.py – Web scraping script
csv_cleaning.ipynb – Data cleaning
internship_analysis.ipynb – EDA
final_internship_data.csv – Processed dataset
Tableau Dashboard file
📊 Dashboard Highlights
Total Internship Opportunities
% Remote vs In-Office Roles
Average Monthly Stipend
City-wise & Category-wise Analysis
Interactive filtering & KPI updates
🔎 Key Insights
~49% internships are remote
Data Science & AI roles dominate
Mumbai & Bangalore show highest demand
Stipend varies significantly by location
💡 Outcome
Built a complete data pipeline from web scraping to visualization, demonstrating real-world data collection, cleaning, and dashboard development skills.
